## 2025-02-11 - Tactile Feedback for Virtual Controls
**Learning:** Virtual buttons and game events often feel disconnected on mobile or with touch-based interaction. Adding subtle haptic feedback (selection clicks for UI and light/medium impacts for game events) provides a sense of physicality and weight, making the interface feel more responsive and delightful.
**Action:** Always consider adding `HapticFeedback` to important state changes or feedback loops (audio-visual cues) to enhance the overall sensory experience.
