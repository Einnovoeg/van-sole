## 2026-03-18 - Tactile and Accessible Controls
**Learning:** Enhancing virtual controls with both 'HapticFeedback' and 'Semantics' provides a more responsive and accessible experience, especially for interactive elements like custom buttons and steppers that lack default platform-level feedback.
**Action:** Always wrap custom interactive widgets with 'Semantics' and trigger 'HapticFeedback' on pointer events to ensure tactile confirmation and accessibility for all users.
