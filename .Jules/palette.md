## 2024-11-06 - Haptic Feedback for Virtual Controls
**Learning:** Virtual touch controls lack the tactile feedback of physical buttons, which can lead to uncertainty in user input, especially in fast-paced games. Integrating haptic feedback provides immediate physical confirmation of an action.
**Action:** Always integrate 'HapticFeedback' (e.g., lightImpact, mediumImpact, selectionClick) to provide tactile confirmation for virtual controls and significant game events on supported devices.
