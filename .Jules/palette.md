# Palette UX Journal

## 2025-05-14 - Haptic Feedback and Accessibility for Virtual Controls
**Learning:** Virtual controls lack the tactile response of physical buttons, which can lead to a disconnected feeling in high-action games like space RPGs. Integrating haptic feedback for game events and UI interactions, combined with semantic labels, significantly improves both the "feel" and the accessibility of the interface.
**Action:** Always pair virtual game controls with `HapticFeedback` and ensure all icon-only buttons use `Semantics` or `Tooltip` with clear labels.
