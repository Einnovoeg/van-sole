## 2026-03-21 - [Sensory Reinforcement with Haptics]
**Learning:** Adding haptic feedback (tactile response) alongside audio cues provides a richer, more immersive sensory experience, especially in a game where physical feedback for actions like firing or being hit is expected.
**Action:** Use 'HapticFeedback' classes (lightImpact, mediumImpact, selectionClick) to reinforce key interactions and state changes across the UI.
