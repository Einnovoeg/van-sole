# Agent Guide: Van Solè

## Project Purpose
Van Solè is a cross-platform spacefaring action RPG inspired by SolarWinds. It features a custom pseudo-3D rendering engine, tactical combat, and an economic simulation.

## What has been done
- **Expanded Universe**: Increased sectors from 3 to 5, adding unique stations, resources, and lore.
- **Combat Enhancements**: Added a homing missile system and Elite Pirate ships with increased difficulty and rewards.
- **RPG Progression**: Implemented Ship Specializations (Combat, Exploration, Trade, Survival) that provide passive bonuses.
- **Visual Upgrades**: Added nebula clouds, planetary rings, and atmospheric glows to the background.
- **Campaign Expansion**: Added new mission types and goals, including anomaly investigation and elite bounty hunting.
- **Economy**: Expanded the commodity catalog with high-value items like Quantum Cores and Relic Parts.
- **Cleanup**: Removed PII, added a proper MIT license, and established a clean project structure.

## Next Steps
1. **AI Overhaul**: Implement more complex tactical behavior for pirate ships (e.g., flanking, retreat).
2. **UI Polish**: Refactor the cockpit UI for a more modern, diegetic feel.
3. **Performance Optimization**: Optimize the depth-sorting and mesh rendering for higher frame rates.
4. **Advanced Questing**: Implement branching dialogue and multi-stage quest lines.
5. **Audio Integration**: Replace basic system sounds with a full atmospheric soundscape.
